## **Library Management System (Python)** 

## **Overview** 

This is a simple **Library Management System** developed using **Python** and **JSON** file handling. 

The project allows users to: 

- Login using username and password 

- Add new books 

- Search books 

- Count books author-wise 

- Remove books 

- Issue books 

- Store data permanently using JSON 

## **Features** 

## **1. Login System** 

Only authorized users can access the library. 

Default credentials: 

- Username: `Abhay` • Password: `1803` 

## **2. Add Book** 

Users can add books with: 

- Book Name 

- Author Name 

Book data is stored in `Library.json` . 

## **3. Author Wise Book Count** 

Shows total number of books available for each author. 

Example: 

1 

```
Author wise book count:
John = 3 books
Harry = 2 books
```

## **4. Search Book** 

Two search options: 

- Search by Author Name 

- Search by Book Name 

## **5. Remove Book** 

Two remove options: 

- Remove a single book 

- Remove all books of an author 

## **6. Clear All Data** 

Deletes all books from the library after confirmation. 

## **7. Issue Book** 

Issue a book to a student. 

Stores: 

- Student Name 

- Issue Date 

- Status 

Example: 

```
Status: issued
Issued To: Rahul
Issued Date: 13-06-2026 10:30 PM
```
